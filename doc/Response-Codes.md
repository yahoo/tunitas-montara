# Response Codes

## Success

* `200`
 * OK (Everything is good)
* `202`
 * Accepted (The document is accepted)

## Redirection

The 300-series codes are not used for signalling.

## Application Failure

* `400`
 * Incomplete query
 * Inappropriate query
* `404`
 * Invalid URL

## Service Failures

* `500`
 * Inappropriate HTTP verb
 * Inappropriate URL
 * Some other problem
